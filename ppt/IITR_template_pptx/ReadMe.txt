User Guidelines:
================
1. The template is named as IITR_PPT_Template, which you can include to your Microsoft PowerPoint software.

2. It contains FIVE different layouts of slides - Title Slide, Content Slide, Comparision Slide, Blank Slide and Last Slide.

3. Please open this PPTX file in your Microsoft PowerPoint software and click on the "New Slide" with a small drop-down arrow just below the Menu “Insert”. Then you can choose any one among the five different types of layouts in this template according to your requirement.

4. For any further help, please contact Dr. Sudip Roy (E-mail ID: sudiproy.fcs@iitr.ac.in).
